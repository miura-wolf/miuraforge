## 🎨 SEMANA 3: THUMBNAILS VERTICALES

### Especificaciones CORREGIDAS

**Tamaño:** 1080×1920px (9:16 vertical) ✅  
**NO 1280×720 (ese es para videos horizontales)** ❌

**Estructura visual:**
```
┌─────────────────┐  ← 20% del espacio
│  TEXTO GRANDE   │     "TE CONVENCISTE"
│   (blanco)      │     "DE QUE"
├─────────────────┤
│                 │  ← 60% del espacio
│   IMAGEN        │     Cara expresiva
│   IMPACTANTE    │     Emoción fuerte
│                 │     (miedo, enojo, shock)
├─────────────────┤
│   LOGO/MARCA    │  ← 20% del espacio
│   (pequeño)     │     Icono de acero
└─────────────────┘
```

**Colores:**
- Fondo: Negro profundo (#0a0a0a) o degradado oscuro
- Texto: Blanco puro (#ffffff) o acero (#c0c0c0)
- Acento: Naranja quemado (#d4561a) - usado mínimo

**Tipografía:**
- Bebas Neue (bold, industrial)
- Tamaño: Grande, legible en móvil
- Máximo 5-6 palabras

**Ejemplos de texto:**
```
"TE CONVENCISTE DE QUE"
"ERES PRODUCTIVO"

"SIGUES ESPERANDO"
"EL MOMENTO PERFECTO"

"TE MIENTES"
"CADA MAÑANA"
```
