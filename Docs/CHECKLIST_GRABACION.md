## 1. EQUIPO NECESARIO

### ✅ Lo Básico (Mínimo Viable)
| Equipo | Recomendación | Alternativa |
|--------|---------------|-------------|
| **Cámara** | Smartphone con cámara 1080p+ | Webcam externa 1080p |
| **Audio** | Micrófono de solapa/corbata ($20-50) | Audífonos con micrófono |
| **Iluminación** | Ring light ajustable ($30-60) | Ventana con luz natural |
| **Fondo** | Pared limpia/color sólido | Cortina neutra |

### 🎯 Lo Ideal (Nivel Pro)
- **Cámara**: DSLR/Mirrorless con lente 50mm o webcam 4K
- **Audio**: Micrófono de condensador USB (Blue Yeti, Røde NT-USB)
- **Iluminación**: Kit 3 puntos (key, fill, back) o softbox
- **Fondo**: Fondo verde para chroma o escenario diseñado

## 2. CHECKLIST PRE-GRABACIÓN

### 📋 Configuración Técnica
- [ ] **Script escrito y revisado** - Guion completo con timestamps
- [ ] **Thumbnail bocetado** - Concepto visual definido antes de grabar
- [ ] **Iluminación lista** - Sin sombras duras en el rostro
- [ ] **Audio probado** - Grabar prueba de 10 segundos y escuchar
- [ ] **Batería cargada** - 80% mínimo en cámara y micrófono
- [ ] **Fondo limpio** - Sin elementos distractores
- [ ] **Modo avión activado** - Evitar notificaciones/interrupciones
- [ ] **Espacio libre** - 10GB+ disponibles para grabación

### 🧠 Preparación Mental
- [ ] **Estado de ánimo positivo** - Energía alta antes de grabar
- [ ] **Voz calentada** - Hacer ejercicios vocales 2-3 minutos
- [ ] **Ropa lista** - Colores sólidos, evitar rayas/logos
- [ ] **Agua a mano** - Hidratado antes y durante

---

## 3. CHECKLIST DURANTE LA GRABACIÓN

### 📹 Configuración de Cámara
- [ ] **Cámara frontal** - Ver tu expresión mientras grabás
- [ ] **Frame correcto** - Pecho para arriba, cabeza con margen arriba
- [ ] **Ojos a cámara** - No mirar la pantalla, mirar EL LENTE
- [ ] **Estabilidad** - Trípode o superficie firme (sin temblores)
- [ ] **Resolución** - 1080p mínimo, 60fps si es posible

### 🎭 Performance
- [ ] **Expresión facial fuerte** - Sonreído, cejas arriba, energía 200%
- [ ] **Gestos naturales** - Usar manos para énfasis
- [ ] **Voz energética** - Más alto y animado de lo normal
- [ ] **Pausas estratégicas** - Dejar silencios para cortes
- [ ] **Múltiples tomas** - Grabar cada frase 2-3 veces

### 🔊 Audio
- [ ] **Nivel correcto** - Picos entre -12dB y -6dB
- [ ] **Sin eco** - Ambiente tratado o acústica controlada
- [ ] **Sin ruido de fondo** - Ventiladores, autos, construcción
- [ ] **Distancia correcta** - Micrófono a 15-20cm de la boca

---

## 4. CHECKLIST DE EDICIÓN

### ✂️ Ritmo y Cortes
- [ ] **Cortes rápidos** - 3-5 segundos máximo por toma
- [ ] **Eliminar pausas** - Quitar "ehm", "aaa", silencios muertos
- [ ] **Jump cuts naturales** - Transiciones suaves entre segmentos
- [ ] **B-roll incluido** - Imágenes de soporte para momentos clave
- [ ] **Zooms tácticos** - Zoom in en momentos de énfasis

### 📝 Subtítulos (Estilo Hormozi)
- [ ] **Texto grande** - 60-80px, fuente bold
- [ ] **Contraste alto** - Blanco con borde negro o fondo oscuro
- [ ] **Palabra por palabra** - Sincronización precisa
- [ ] **Énfasis visual** - Colores/emojis en palabras clave
- [ ] **Posición inferior** - No tapar el rostro

### 🎵 Audio Final
- [ ] **Música de fondo** - Volumen bajo (-20dB a -25dB)
- [ ] **Loop continuo** - Sin cortes bruscos en la música
- [ ] **Normalización** - Nivel consistente de voz
- [ ] **Reducción de ruido** - Eliminar hiss/fondo si es necesario

### 🎯 Elementos de Conversión
- [ ] **Hook inicial** - Primeros 3 segundos captan atención
- [ ] **CTA claro** - "Seguime", "Guardá este video", "Compartí"
- [ ] **Branding** - Logo/watermark discreto
- [ ] **Duración óptima** - 60-90 segundos (shorts/reels)

## 💡 TIPS PROFESIONALES

### Iluminación
```
✅ ANGULO IDEAL: 45° arriba y al costado (foco principal)
✅ DIFUSION: Si es luz dura, usar cortina blanca o papel manteca
✅ TEMPERATURA: 5600K (luz día) para tonos naturales
✅ EVITAR: Luz desde abajo (efecto Halloween), luz detrás (silueta)
```

### Ángulos de Cámara
```
✅ OJO A OJO: Cámara a la altura de tus ojos (natural y conecta)
✅ LIGERO LOW ANGLE: Cámara 10-15cm abajo (poder, autoridad)
⚠️  HIGH ANGLE: Evitar (disminuye presencia, viste "amateur")
✅ DISTANCIA: Brazo extendido o 50-80cm del rostro
```

### Configuración de Cámara (Smartphone)
```
RESOLUCIÓN: 1080p o 4K
FPS: 30fps (estándar) / 60fps (si hay movimiento)
ESTABILIZACIÓN: Activada
HDR: Activado (mejora contraste)
FORMATO: MP4 (universal)
```
